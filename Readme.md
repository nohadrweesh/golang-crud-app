# Description
- This Application is from this [tutorial](https://www.youtube.com/watch?v=TkbhQQS3m_o)
- This is a CRUD app where we can view/create/update/delete movies using any rest client like postman

## How to run the app:
- Clone the repo
- Cd into the app
- Run these command 
  ```
  go build
  go run main.go
  ```
- The app has 4 routes:
    - GET getMovies
    - GET getMovie/{id}
    - POST getMovie
    - PUT  getMovie/{id}
    - DELETE  getMovie/{id}
